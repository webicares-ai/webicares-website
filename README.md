# Webicares Digital Agency Website

Static HTML/CSS/JS website — no build step required.

## Pages
- `index.html` — Homepage
- `webicares-pricing.html` — Pricing
- `services-*.html` — 4 service pages (SEO, Google Ads, Meta Ads, Web Development)
- `case-study-*.html` — 4 client case studies
- `portfolio.html`, `about.html`, `contact.html`
- `legal-*.html` — Privacy, Terms, Cookies, Disclaimer, Refund
- `webicares-style.css` — shared stylesheet used by all inner pages

## Deploy
Connect this repo to Cloudflare Pages — no build command needed, output directory is `/` (root).
